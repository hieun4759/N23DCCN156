## 📂 Nội dung thực hiện

- **Chương 9:** Server Actions & Form Validation  
  - Xây dựng form gửi dữ liệu lên server.  
  - Xử lý dữ liệu với Server Actions (Next.js / Node.js hoặc công nghệ tương đương).  
  - Validate form phía client (HTML5, JavaScript) và phía server (tùy chọn dùng thư viện như Zod,…).  
  - Hiển thị thông báo lỗi, trạng thái submit và kết quả xử lý.

## 🚀 Cách xem và chạy sản phẩm

1. Clone repo về máy:
   ```bash
   git clone https://github.com/hieun4759/N23DCCN156.git
   ```

2. Di chuyển vào thư mục dự án:
   ```bash
   cd N23DCCN156
   ```

3. Mở bằng VS Code hoặc IDE bạn dùng:
   ```bash
   code .
   ```

4. Với các bài thuần HTML/CSS/JS:
   - Mở trực tiếp file `index.html` bằng trình duyệt, hoặc  
   - Sử dụng extension **Live Server** trong VS Code để chạy nhanh.

5. Với bài có backend (Node.js / Next.js / PHP / …):
   - Xem hướng dẫn chi tiết trong README của từng thư mục bài:
     - Cài đặt dependency (`npm install`, `composer install`, …).
     - Chạy server (`npm run dev`, `npm start`, …).
     - Cấu hình biến môi trường, kết nối database (nếu có).
